weatherman.js

a 2KB uncompressed dependency-free library by Derek Howard for getting the weather from OpenWeatherMap (you'll need an API key)

Download

How minimal is it?

You can start interacting with weather data after one line of code:

var weatherman = new Weatherman('YOUR_API_KEY_HERE', 'someplace', 'imperial/metric', callback);

After that, you can interact with the data in two ways.

  1. Using the weather object (must wait for AJAX to complete):

    alert(weatherman.weather.temperature);

  2. Using the argument passed to the callback:

    function callback(weather) {
    alert(weather.humidity);
}

Full API

function callback(weather) {
    weather.conditions;      //one word description of current weather (clear, clouds, snow, rain)
    weather.description;     //brief description of what's going on
    weather.temperature;     //current temperature (units chosen in constructor)
    weather.humidity;        //current humidity
    weather.wind.speed;      //wind speed in mps
    weather.wind.direction;  //wind direction in dergees
    weather.cloudCoverage;   //percentage of sky covered in cluds
    weather.sunrise;         //Date() object with time of sunrise
    weather.sunset;          //Date() object with time of sunset
}
//create the object
var weatherman = new Weatherman('YOUR_API_KEY_HERE', 'someplace', 'imperial/metric', callback)
//re-fetch the weather
weatherman.retrieveWeather();

Example Usage

Weather in Columbia, MO: 

//function to show the data given by Weatherman
function weatherDisplay(weather) {
    document.getElementById('conditions').innerHTML = weather.conditions + ', ' + weather.temperature + '°F';
}
//create a Weatherman object
var weatherman = new Weatherman('YOUR_API_KEY_HERE', 'Columbia, MO', 'imperial', weatherDisplay);
